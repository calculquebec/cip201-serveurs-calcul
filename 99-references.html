Documentation Slurm

  • Commande sbatch : pour soumettre une tâche

  • Commande squeue : pour voir l’état des tâches gérées par l’ordonnanceur

  • Commande sacct : pour obtenir de l’information sur les tâches passées

  • Commande sshare : pour l’état de l’utilisation du compte de calcul

  • Commande scontrol show job : pour obtenir les spécifications d’une tâche, dont la partition

  • Commande scontrol show hostname : pour obtenir la liste des noeuds réservés à partir d’une tâche
